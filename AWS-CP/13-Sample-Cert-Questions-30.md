# Sample Certification Questions (30)

1) Which AWS service provides object storage?  
A) EBS  B) EFS  C) S3  D) FSx  
**Answer: C**

2) Which service is best for running containers without managing servers?  
A) EC2  B) ECS on Fargate  C) EKS on EC2  D) Lightsail  
**Answer: B**

3) Which AWS service provides DNS routing?  
A) CloudFront  B) Route 53  C) VPC  D) API Gateway  
**Answer: B**

4) What provides highly available relational databases with automated backups?  
A) DynamoDB  B) RDS  C) S3  D) ElastiCache  
**Answer: B**

5) Which service helps with centralized account management and SCPs?  
A) IAM  B) Organizations  C) Config  D) CloudTrail  
**Answer: B**

6) Which option provides lowest-cost long-term archival storage?  
A) S3 Standard  B) S3 IA  C) Glacier Deep Archive  D) EBS  
**Answer: C**

7) Which is a key benefit of CloudFront?  
A) Database replication  B) CDN caching  C) IAM role creation  D) VPC routing  
**Answer: B**

8) Which service provides managed message queues?  
A) SNS  B) SQS  C) EventBridge  D) MQ  
**Answer: B**

9) For HA in RDS, which feature is used?  
A) Read Replica  B) Multi-AZ  C) Snapshot  D) DAX  
**Answer: B**

10) Which is a best practice for IAM?  
A) Use root for daily tasks  B) Enable MFA  C) Share access keys  D) Grant admin to everyone  
**Answer: B**

11) Which service provides audit logs of AWS API calls?  
A) CloudWatch  B) CloudTrail  C) Config  D) GuardDuty  
**Answer: B**

12) Which service is best for a fully managed key-value store?  
A) RDS  B) DynamoDB  C) ElastiCache  D) Redshift  
**Answer: B**

13) Which service is a managed NFS file system?  
A) EBS  B) EFS  C) S3  D) FSx for Windows  
**Answer: B**

14) Which service provides application-level load balancing?  
A) NLB  B) ALB  C) GWLB  D) CloudFront  
**Answer: B**

15) Which tool estimates monthly AWS costs?  
A) Budgets  B) Cost Explorer  C) Pricing Calculator  D) Trusted Advisor  
**Answer: C**

16) Which service provides serverless compute?  
A) EC2  B) Lambda  C) ECS  D) EKS  
**Answer: B**

17) Which service is used to store secrets with rotation?  
A) Systems Manager Patch  B) Secrets Manager  C) Config  D) CloudFormation  
**Answer: B**

18) Which feature allows private access to S3 without internet?  
A) NAT Gateway  B) VPC Endpoint  C) IGW  D) Route 53  
**Answer: B**

19) Which service provides centralized security findings?  
A) Security Hub  B) CloudTrail  C) Inspector  D) CloudWatch  
**Answer: A**

20) Which service is best for streaming data ingestion?  
A) Glue  B) Kinesis  C) Redshift  D) Athena  
**Answer: B**

21) Which option is cheapest for fault-tolerant batch jobs?  
A) On-Demand EC2  B) Reserved Instances  C) Spot Instances  D) Dedicated Hosts  
**Answer: C**

22) Which service provides centralized configuration compliance?  
A) Config  B) CloudTrail  C) IAM  D) S3  
**Answer: A**

23) Which AWS service is a CDN?  
A) Route 53  B) CloudFront  C) Direct Connect  D) Global Accelerator  
**Answer: B**

24) Which tool is used to build IaC templates?  
A) CloudFormation  B) Systems Manager  C) OpsWorks  D) Elastic Beanstalk  
**Answer: A**

25) Which service provides DDoS protection?  
A) GuardDuty  B) Shield  C) WAF  D) Macie  
**Answer: B**

26) Which service manages Kubernetes?  
A) ECS  B) EKS  C) Fargate  D) Batch  
**Answer: B**

27) Which database is best for OLAP analytics?  
A) RDS  B) Redshift  C) DynamoDB  D) Aurora  
**Answer: B**

28) Which feature improves EC2 resilience automatically?  
A) Spot Fleet  B) Auto Scaling  C) Placement Groups  D) Dedicated Hosts  
**Answer: B**

29) Which service provides a managed API gateway?  
A) ALB  B) API Gateway  C) NLB  D) Route 53  
**Answer: B**

30) Which service stores data blocks for EC2?  
A) S3  B) EBS  C) EFS  D) FSx  
**Answer: B**
